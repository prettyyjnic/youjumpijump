## 使用须知

Android 设备

1. Android 手机一台，电脑上安装 ADB，连接上电脑后开启 USB 调试模式，如果有模拟点击的选项请一并开启
2. 进入微信打开微信跳一跳，点击开始游戏
3. 解压包 微信跳一跳助手v1.0.zip
3. 双击 main.exe

iOS 设备
